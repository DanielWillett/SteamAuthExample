This repository serves as an example usage of my [SteamOid2](https://www.nuget.org/packages/SteamOid2) library.

It was created for a ITCS-3166 (Systems and Networks) class I'm taking to introduce a basic example of using the deprecated OpenID 2.0 protocol to authenticate with an Identity Provider (Steam) for Single Sign-On.

![auth_graph](https://raw.githubusercontent.com/DanielWillett/SteamAuthExample/refs/heads/master/assets/auth_graph.png)

Main stuff is happening in:
* [SteamLoginHome.razor](https://github.com/DanielWillett/SteamAuthExample/blob/master/Components/Pages/SteamLoginHome.razor) 
* [SteamLoginVerification.razor](https://github.com/DanielWillett/SteamAuthExample/blob/master/Components/Pages/SteamLoginVerification.razor)
